# Revenue streams

## Overview

Six streams, ordered from fastest to slowest to activate. All feed from the same content engine.

---

## Stream 1 — Artist website builds
**Timeline:** Fastest — can start this week
**Price:** $500–$2,000 per site
**Platform:** Instagram, Facebook, Meta ads — NOT LinkedIn

### Tiers
| Tier | Price | What's included |
|---|---|---|
| Starter | $500–$800 | WordPress setup, 5 pages, contact form, Instagram embed, basic SEO |
| Standard | $1,000–$1,500 | Everything + WooCommerce shop + Teachery integration + payment gateway |
| Premium | $2,000–$3,000 | Full site + 3 months support + 2hr training + marketing setup |

### How to get first clients
1. Ask Debasree to refer one artist friend — first client at cost for the testimonial
2. Post on Instagram/Facebook offering the service — use debasreedeyart.com as reference
3. Document every build publicly — each post is a lead magnet for the next client
