**Update frequency:** Daily or whenever inspiration hits

**Purpose:** Capture post ideas BEFORE they're forgotten. Gold mine for LinkedIn/X/Instagram content.

### How to use:

1. See something interesting? Jot it down here.
2. Read a cricket stat? Add it.
3. Build something and think of a different angle? Add it.
4. See what others post and think "I could do that better"? Add it.
5. Every Friday, pick 1-2 ideas to expand into full posts.

### Template for each idea:

```
**[Date] — [Potential title or hook]**
- Angle: [What makes this interesting]
- Platform: LinkedIn / X / Instagram / All
- Why now: [What triggered the idea]
- Rough outline: [Bullet points if you have them]
- Status: [Idea / Ready to write / In progress / Published]
```
