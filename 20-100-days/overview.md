# 100 days project — overview

## What it is

A public commitment to build 100 small, useful things over 100 days — documenting every single one across your website, LinkedIn, X, Instagram, and YouTube.

**Start date:** TBD
**End date:** TBD (100 days from start)
**Hashtag:** #100DaysOfBuilding

---

## What counts as one "build"

- A working script or tool (even 50 lines of Python)
- A data finding with a chart
- A new feature or automation
- A working prototype — rough but functional
- A Cursor prompt template or workflow
- A dashboard or visualisation
- A website feature or component

**Rule:** It must be real and runnable. Not a plan. Something you can show working.

|---|---|---|---|

| Cricket analytics | 11–30, 41–60, 71–90 | LinkedIn + X + YouTube | LinkedIn, X, YouTube |
| Artist + web tools | 21–30, 51–60, 91–100 | Artists | Instagram, Facebook |

---

## The content multiplication rule

Every build produces:
1. LinkedIn post (10 min)
2. X/Twitter thread (5 min — trim from LinkedIn)
3. Blog post on website (15 min — expand from LinkedIn)
4. YouTube video (every other weekend — record during build)

**One build = four pieces of content.**
