## Decision rules

**When you don't know what to work on:**

1. Is there an unfinished build? → Finish and ship it first
2. Is there an unwritten post from something you've done? → Write it
3. Is there a client or consultancy enquiry waiting? → That pays, prioritise it
4. Is there a half-made decision blocking everything else? → Make it now, move on

**When you feel overwhelmed:** Post before it's perfect. Build before you have a plan. Document before it's polished. The messy start is the content.
