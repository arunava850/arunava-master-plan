## Decision rules

**When you don't know what to work on next:**
1. Is there an unwritten post from a build you've done? → Write it first
2. Is there a cricket dataset you haven't explored? → Explore it and post about it
3. Is there an artist website client waiting? → That pays, prioritise it
4. Do you have a half-finished build? → Finish and ship it

**When you feel overwhelmed:**
Post before it's perfect. Build before you have a plan. Document before it's polished. The messy start is the content.
